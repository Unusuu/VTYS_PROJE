# Veritabanı Şeması

Bu klasör veritabanı şema dosyalarını içerir.

## Dosyalar

- `setup_database.sql` - Veritabanı kurulum scripti

## Tablolar

| Tablo | Açıklama |
|-------|----------|
| books | Kitap bilgileri |
| copies | Kitap kopyaları |
| members | Üye bilgileri |
| loans | Ödünç kayıtları |
| loan_history | Ödünç geçmişi |

## Kurulum

SQL Server Management Studio'da `setup_database.sql` dosyasını çalıştırarak veritabanını oluşturabilirsiniz.
