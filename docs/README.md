# Dokümantasyon

Bu klasör proje dokümantasyonunu içerir.

## İçerik

- API Dokümantasyonu
- Kullanım Kılavuzu
- Geliştirici Notları

## Proje Hakkında

KutuphaneOtomasyonu, modern bir kütüphane yönetim sistemidir. ASP.NET Core 10 ve Entity Framework Core kullanılarak geliştirilmiştir.

### Temel Özellikler

1. **Kitap Yönetimi** - Kitap ekleme, düzenleme, silme
2. **Üye Yönetimi** - Üye kayıt, güncelleme, durum yönetimi
3. **Ödünç İşlemleri** - Kitap ödünç verme ve iade
4. **Raporlama** - Popüler kitaplar, üye istatistikleri
